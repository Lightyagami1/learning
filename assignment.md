Assignmet 1:
------------------------

Write a menu driven program with options to create, retrieve, update and delete (CRUD operation) for contact list. Contacts will contain name, number, address etc. details. You are required to use JSON based file write and read.

Assignmet 2:
------------------------
Same as assignment 1 but instead of using file based JSON read/write use JDBC connection to save the contact details in Postgres SQL.


Assignment 3
------------------------
Using Jersey, make the following API endpoints

1.	/api/contacts/get?name=value
2.	/api/contacts/delete?name=value
3.	/api/contacts/put?name=value&phone=value
4.	/api/contacts/post?name=value&phone=value
5.	/api/contacts/reset
